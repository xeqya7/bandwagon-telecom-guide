## 一、电信用户为什么特别适合搬瓦工？

CN2（中国电信 Next Generation Network）是电信自己建的高端国际专线，分两个档次：CN2 GT（普通版）和 CN2 GIA（高端版，也叫 Global Internet Access）。

CN2 GIA 和普通线路最大的区别不在于速度，而在于**高峰期的抗拥堵能力**。普通线路晚 8-11 点是真的堵，掉速掉得厉害；CN2 GIA 因为走的是专用带宽，绕开了拥堵的公共骨干，高峰期表现要稳得多。

搬瓦工把几个核心机房——DC6、DC9、香港 HKHK_8、日本东京 JPTYO_8——全部接入了 CN2 GIA 线路。对电信用户来说，这些机房去程回程都走电信自家的 59.43 节点（AS4809），不需要经过联通或移动的骨干中转，路由干净，延迟低。

这是搬瓦工在电信用户群里口碑好的根本原因，不是营销词，就是线路本身的优势。

## 五、CN2 GT 还是 CN2 GIA？电信用户必须选 GIA 吗？

不一定。

CN2 GT 是 CN2 普通版（Global Transit），带宽共享更多，晚高峰容易出现拥堵和丢包。对于用 VPS 做一般建站、偶尔 SSH 登录、流量需求不高的用户，CT GT 日常也够用，价格便宜一截。

问题是，一旦晚高峰你需要靠谱的速度——比如视频传输、大文件上传、需要实时访问——CN2 GT 就撑不住了。这种时候就必须 GIA。

说实话，如果你的主要使用场景是：

- 建站，但流量不大，每天就几十个访客
- 偶尔 SSH 上去跑个脚本
- 对延迟没有严格要求

那 KVM 套餐（$49.99/年，普通线路）完全够用。省下来的钱买别的。

但如果你：

- 用 VPS 做梯子或流量代理
- 跑需要稳定国际连接的服务
- 经常在晚高峰使用
- 建的是有真实访问量的商业站

那 CN2 GIA-E 是必须的。晚高峰那两三个小时的体验差距，用过才知道。
